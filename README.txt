PECS APP (Flutter) — README
===========================

Это минимальное приложение для Android (и потенциально iOS), которое показывает
карточки, озвучивает их текст и позволяет маме добавлять новые карточки,
в том числе через поиск картинок в интернете (Pexels).

ФУНКЦИИ
-------
- Главный экран с карточками (встроенные + добавленные).
- Адаптивная сетка: под телефоны и планшеты (2–4 столбца).
- Нажатие по карточке — голосом читается фраза на русском (Google TTS).
- Кнопка настроек (шестерёнка справа вверху):
  - вход в настройки только по длительному удержанию (2–3 секунды).
- В настройках:
  - просмотр встроенных карточек;
  - список пользовательских карточек (можно удалить);
  - добавление карточки из камеры или галереи;
  - поиск картинок через Pexels и добавление как новой карточки.

ВАЖНО
-----
1. Перед сборкой замените строку с API-ключом Pexels в файле:
   lib/main.dart

   Найдите строку (в конце файла, в классе _PexelsSearchScreenState):

       static const String _pexelsApiKey = 'YOUR_PEXELS_API_KEY_HERE';

   и замените на свой ключ, например:

       static const String _pexelsApiKey = '1234567abcdef...' ;

2. На устройстве должен быть установлен русскоязычный голос TTS
   (обычно это Google Text-to-Speech, язык ru-RU).

-----------------------------------------
УСТАНОВКА И СБОРКА НА LINUX
-----------------------------------------

1. Установите Flutter
   - Откройте официальную инструкцию Flutter для Linux.
   - В целом шаги такие:
     - скачайте архив Flutter SDK;
     - распакуйте, например, в ~/flutter;
     - добавьте в PATH:
       export PATH="$PATH:$HOME/flutter/bin"
     - выполните:
       flutter doctor

2. Установите Android SDK/платформу
   - Можно поставить Android Studio и через него поставить SDK и инструменты.
   - После этого flutter doctor должен показать, что Android toolchain настроен.

3. Подготовьте проект
   - Распакуйте этот ZIP, например, в ~/projects/pecs_app
   - Перейдите в папку проекта:
       cd pecs_app
   - Загрузите зависимости:
       flutter pub get

4. Вставьте Pexels API ключ
   - Откройте файл lib/main.dart в любом редакторе (nano, vim, VS Code).
   - Замените строку с 'YOUR_PEXELS_API_KEY_HERE' на свой ключ.

5. Запуск на подключённом устройстве
   - Включите режим разработчика и отладку по USB на планшете/телефоне.
   - Подключите устройство к компьютеру по USB.
   - Проверьте, что устройство видно:
       flutter devices
   - Запустите приложение:
       flutter run

6. Сборка APK
   - Для сборки release-APK выполните:
       flutter build apk --release
   - Готовый файл будет лежать здесь:
       build/app/outputs/flutter-apk/app-release.apk
   - Перенесите APK на планшет/телефон и установите.

-----------------------------------------
УСТАНОВКА И СБОРКА НА WINDOWS
-----------------------------------------

1. Условие: установлен Git for Windows и PowerShell.

2. Установите Flutter
   - Скачайте Flutter SDK для Windows с официального сайта.
   - Распакуйте, например, в C:\src\flutter
   - Добавьте путь C:\src\flutter\bin в PATH (через "Переменные среды").
   - Откройте новую консоль PowerShell и выполните:
       flutter doctor

3. Установите Android Studio / SDK
   - Поставьте Android Studio, установите Android SDK и необходимые компоненты.
   - Откройте команду:
       flutter doctor
     и убедитесь, что "Android toolchain" в порядке.

4. Подготовьте проект
   - Распакуйте ZIP в папку, например: D:\projects\pecs_app
   - В PowerShell:
       cd D:\projects\pecs_app
       flutter pub get

5. Вставьте Pexels API ключ
   - Откройте файл lib\main.dart в любом редакторе (VS Code, Notepad++).
   - Найдите 'YOUR_PEXELS_API_KEY_HERE' и замените на настоящий ключ.

6. Запуск на устройстве или эмуляторе
   - Включите USB-отладку на телефоне и подключите его.
   - Либо запустите эмулятор через Android Studio.
   - Проверка устройств:
       flutter devices
   - Запуск:
       flutter run

7. Сборка APK
   - В PowerShell из корня проекта:
       flutter build apk --release
   - APK будет здесь:
       build\app\outputs\flutter-apk\app-release.apk
   - Скопируйте файл на устройство и установите.

-----------------------------------------
ИСПОЛЬЗОВАНИЕ ПРИЛОЖЕНИЯ
-----------------------------------------

1. Первый запуск
   - Сразу откроется экран с карточками.
   - Встроенные карточки уже доступны (кушать, пить, гулять, мама и т.д.).

2. Для ребёнка
   - Ребёнок просто нажимает на карточку.
   - Телефон/планшет произносит фразу голосом.

3. Вход в режим мамы
   - В правом верхнем углу экрана есть значок шестерёнки.
   - Нажмите и удерживайте примерно 2–3 секунды.
   - Появится индикатор прогресса по кругу.
   - После полного заполнения откроются настройки.

4. Добавление своей карточки (камера/галерея)
   - В настройках нажмите:
       "Добавить карточку (камера/галерея)"
   - Выберите источник изображения.
   - Сделайте фото или выберите из галереи.
   - Введите текст на карточке и фразу для произношения.
   - Сохраните — карточка появится на главном экране.

5. Добавление карточки через поиск в интернете (Pexels)
   - В настройках нажмите:
       "Найти картинку в интернете (Pexels)"
   - Введите, например: "яблоко", "печенье", "игрушка", "сок".
   - Нажмите на найденную картинку.
   - Введите текст и фразу.
   - Сохраните — карточка добавится в общий список.

-----------------------------------------
ПРАКТИЧЕСКИЕ ПОДСКАЗКИ
-----------------------------------------

- Если озвучка не работает:
  - Проверьте, что на устройстве установлен Google Text-to-Speech.
  - В настройках системы выберите русский голос.
- Если Pexels не работает:
  - Проверьте правильность API-ключа.
  - Проверьте наличие интернета.
- Встроенные карточки (assets/preset/*.png) можно заменить своими
  рисунками/пиктограммами, сохранив те же имена файлов.

Если что-то не собирается или будет ошибка при "flutter run" —
можно прислать лог ошибки, и тогда уже точечно разбираться.
